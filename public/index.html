Lisp Interpreter

 

Examples:

Arithmetic
  • (+ 2 3) => 5
  • (+ 2 (* 5 6)) => 32
  • (+ (+ 2 (+ 1 2)) (* (+ 1 2) (+ 1 2)))
       => (+ 5 9)
       => 14
Variables
  • (set! somevar 5) => true (success)
  • (defined somevar) => somevar is 5; true
Anonymous and named functions
  • (4 (lambda n (+ n 3))) => 7
  • (define myfunc (lambda n (* n 4))) => true
  • (myfunc 5) => 20
Conditionals
  • (if (= 1 2) (3) (4)) => 4
  • (/ 20 (if (= (* 7 7) (49)) (2) (5)))
       => (/ 20 (if (= (49) (49)) (2) (5)))
       => 10
